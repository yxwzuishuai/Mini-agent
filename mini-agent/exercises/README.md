# 🏋️ Mini Agent 拆解练习

按顺序完成以下 6 个练习，每个练习聚焦一个知识点。
完成后你就能从零手写一个完整的 Agent。

每个练习文件里有详细的提示和要求，写完后运行测试看看对不对。

## 练习顺序

| 练习 | 文件 | 知识点 | 预计时间 |
|------|------|--------|----------|
| 1 | `ex1_dict.py` | 字典操作 + `**` 解包 | 10 分钟 |
| 2 | `ex2_env.py` | 环境变量 + dotenv | 10 分钟 |
| 3 | `ex3_function.py` | 函数定义 + 类型提示 | 15 分钟 |
| 4 | `ex4_class.py` | 类 + `__init__` + `self` | 15 分钟 |
| 5 | `ex5_tools.py` | 工具定义 + 注册表模式 | 20 分钟 |
| 6 | `ex6_agent.py` | 完整 Agent Loop | 30 分钟 |

## 使用方法

```bash
cd mini-agent/exercises

# 写完一个练习后运行它
python ex1_dict.py
python ex2_env.py
# ...以此类推
```

每个文件底部都有自测代码，运行后会告诉你对不对。
遇到卡住的地方，回去看 `mini-agent/` 下的源码对照。
