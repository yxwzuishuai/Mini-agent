Python 基础知识

Python 是一种高级编程语言，由 Guido van Rossum 于 1991 年创建。
Python 的设计哲学强调代码的可读性和简洁性。

Python 的主要特点：
1. 语法简洁，易于学习
2. 支持多种编程范式：面向对象、函数式、过程式
3. 拥有丰富的标准库和第三方库
4. 动态类型，不需要声明变量类型
5. 自动内存管理（垃圾回收）

Python 常用数据结构：
- 列表（list）：有序可变序列，用方括号 [] 表示
- 元组（tuple）：有序不可变序列，用圆括号 () 表示
- 字典（dict）：键值对集合，用花括号 {} 表示
- 集合（set）：无序不重复元素集合

Python 的应用领域：
- Web 开发（Django、Flask、FastAPI）
- 数据科学和机器学习（NumPy、Pandas、Scikit-learn）
- 人工智能（TensorFlow、PyTorch）
- 自动化脚本和运维
- 爬虫和数据采集
