AI Agent 技术介绍

AI Agent（智能体）是一种能够自主感知环境、做出决策并采取行动的人工智能系统。

Agent 的核心组成：
1. 大语言模型（LLM）：Agent 的"大脑"，负责理解和推理
2. 工具（Tools）：Agent 的"手脚"，用于执行具体操作
3. 记忆（Memory）：Agent 的"记忆"，存储对话历史和知识
4. 规划（Planning）：Agent 的"思考过程"，决定下一步做什么

Agent 的工作流程（Agent Loop）：
1. 接收用户输入
2. LLM 分析需求，决定是否需要使用工具
3. 如果需要工具，调用工具并获取结果
4. 将工具结果返回给 LLM
5. LLM 根据结果生成最终回复
6. 返回给用户

常见的 Agent 框架：
- LangChain / LangGraph：最流行的 LLM 应用框架
- CrewAI：专注多 Agent 协作
- AutoGen：微软开发的多 Agent 框架
- Dify：低代码 AI 应用平台

RAG（检索增强生成）：
RAG 是一种让 LLM 能够访问外部知识的技术。
流程：文档切片 → 向量化 → 存入向量数据库 → 用户提问时检索相关内容 → 拼入 prompt → LLM 回答
RAG 的优势是让 LLM 能回答它训练数据中没有的内容，比如公司内部文档、最新资料等。
