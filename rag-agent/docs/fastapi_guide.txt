FastAPI 入门指南

FastAPI 是一个现代、高性能的 Python Web 框架，用于构建 API。
它基于 Starlette（异步 Web 框架）和 Pydantic（数据校验）。

FastAPI 的核心特点：
1. 高性能：与 Node.js 和 Go 相当
2. 自动生成 API 文档：访问 /docs 即可看到 Swagger UI
3. 类型提示驱动：利用 Python 类型提示自动校验请求和响应
4. 异步支持：原生支持 async/await
5. 依赖注入系统：类似 Spring 的 IoC

FastAPI 基本用法：
- 创建应用：app = FastAPI()
- 定义路由：@app.get("/") 或 @app.post("/")
- 请求体：使用 Pydantic BaseModel 定义
- 路径参数：@app.get("/items/{item_id}")
- 查询参数：def read_items(skip: int = 0, limit: int = 10)

FastAPI vs Spring Boot：
- 路由：@app.get ≈ @GetMapping
- 请求体：BaseModel ≈ @RequestBody + DTO
- 参数校验：Pydantic 自动校验 ≈ @Valid + JSR 303
- 启动：uvicorn ≈ Tomcat
- 文档：自动生成 ≈ 需要配置 Swagger
